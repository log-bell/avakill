AvaKill

The AI Safety Firewall.

Intercepts agent tool calls in real-time. Blocks destructive operations before they execute. Protects your databases, files, and infrastructure from rogue AI.

Works with OpenAI, Anthropic, LangChain, MCP, and every major framework.

›Quick Start

macOS/Linux/Windows β BETA
# Protect your agents in one command.
$ pip install avakill && avakill init

Works on macOS, Linux & Windows. Use pipx or a virtualenv on macOS.

›What It Does

Tool-Call Interception

Intercepts every tool call between your AI agent and its tools. Blocks destructive operations before they execute — not after.

Sub-Millisecond Latency

Pure rule evaluation, no ML models. Adds <1ms overhead to tool calls that already take 500ms–5s. Your agents won’t even notice.

YAML Policies

Simple, readable safety rules anyone can write. Deny-by-default, allow lists, rate limits, argument pattern matching. Version-controlled security.

Framework Agnostic

OpenAI, Anthropic, LangChain, LangGraph, MCP. Drop-in wrappers for major frameworks, plus a decorator for any Python function.

Complete Audit Trail

Every tool call logged with decision, policy, latency, and full arguments. SQLite storage, CLI queries, JSON export. Built-in analytics.

MCP Proxy

Drop-in transparent proxy for any MCP server. Intercepts tools/call requests without changing your server or client code.
